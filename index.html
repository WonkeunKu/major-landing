초기에는 핵심 학과부터 시작해, 검수된 데이터로 단계적으로 확장합니다.

학과 이름은 아는데, 실제로 뭘 하는지 모르는 상태에서 선택하잖아요.

이건 “의지 부족”이 아니라, 정보 구조가 잘못된 문제예요.

학생이 겪는 현실
  • 홍보 영상은 멋있는데, 현실이랑 괴리가 큼
  • 상담은 추상적이라 결정 근거가 부족
  • “취업 잘 됨”만 있고, 과정이 안 보임
  • 결국 점수 맞춰 지원 → 뒤늦은 후회
우리가 만드는 해결 방식

전공을 “설명”이 아니라 “체험”으로 보여줍니다.

입학 전, 1학년 1학기부터 실제 과제·수업 흐름을 미리 경험하고 학과별 난이도/과제량/팀플/적합 포인트까지 비교해 내가 버틸 수 있는 전공을 고르게 합니다.

핵심 기능

“체험 깊이”가 생기면, 비교/추천은 자연스럽게 따라옵니다.

1) 전공 시뮬레이션

1학년 1학기 수업 흐름 + 미니 과제 체험

2) 현실 데이터

과제량, 팀플, 시험 난이도, 만족도(대학생)

3) 학과 비교

“뭐가 다른지” 한 장 표로 명확히

4) 개인 리포트

잘 맞는 점/힘든 점/추천 학과 TOP3 PDF

이런 학생에게 잘 맞아요

전공이 너무 많아 결정이 어려운 고1~고2

탐색 단계에서 체험으로 후보를 줄입니다.

문과/이과 갈림길에 있는 학생

과제/수업 기준으로 “맞는 활동”을 찾습니다.

부모/선생님께 설명 근거가 필요한 학생

리포트(PDF)로 상담/면담이 쉬워집니다.

사용 방법

3단계로 구성 '추천 → 체험 → 비교 리포트'

1
관심 분야 선택 (30초)

문과/이과/융합, 선호 활동과 싫어하는 작업을 입력

2
원하는 전공 선택 → 체험

미니 과제/수업 흐름을 직접 해보며 “현실감” 확보

3
비교 리포트 저장/공유

적합 포인트/주의 포인트 정리 → PDF로 공유

신청은 구글폼으로 진행됩니다. (정가 20만원 → 할인가 10만원)

체험을 통해 “나에게 맞는 전공인지”를 미리 확인하고, 결과를 부모/선생님과 공유할 수 있는 리포트(PDF)까지 제공합니다.
